## ⚙️ Proyectos profesionales

### 🛸 DTE Manager v2
Sistema de microservicios para gestión documental tributaria (DTE) en producción en Casino Express. Reemplaza procesos manuales de días por automatización de minutos.

**Stack:** FastAPI · Docker · PostgreSQL · Redis · Celery · Playwright · NGINX
- 7 microservicios orquestados (gateway, frontend, auth, dte, bots, db, redis)
- Frontend con FastAPI + htmx + Tailwind
- Autenticación JWT + RBAC
- Workers asíncronos con Celery para procesamiento batch
- Bots headless con Playwright para extracción de datos

🔗 [v2-dte.kudawa.com](https://v2-dte.kudawa.com)

### 📊 DTE Manager v1 (Streamlit)
Primera versión del sistema — dashboard con KPIs, bots de extracción desde portales externos (i-Construye), normalización de RUTs y limpieza de datos.

**Stack:** Streamlit · SQLite · Playwright · pandas · plotly

🔗 [dte.kudawa.com](https://dte.kudawa.com)

### 🤖 Bot AX Contable
RPA para Microsoft Dynamics AX que registra diarios contables automáticamente mediante reconocimiento de imágenes (OCR + template matching). Interfaz gráfica dual (tkinter/PyQt6), watchdog de monitoreo y +99 iteraciones de mejora continua.

**Stack:** Python · Tesseract OCR · pyautogui · tkinter · PyQt6

🔗 [github.com/kudawasama/Bot_AX_Contable](https://github.com/kudawasama/Bot_AX_Contable)
